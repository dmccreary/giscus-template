# Installing the Feedback and Giscus Comment System on a Mkdocs Intelligent Textbook

This site has detailed step-by-step instructions for installing
both a feedback systems and the Gisus comment management system into a mkdocs system.  Along
with the tutorial it also provides a claude code skill for automating the installation
process.

Note that by default, all our intelligent textbooks have a document edit icon enabled.
This is the easiest way to propose an addition or a change.
